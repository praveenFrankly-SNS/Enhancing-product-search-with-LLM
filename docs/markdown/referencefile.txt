# 2. Evaluation

Evaluation answers:

> "How good is our search system?"

You still need to calculate:

```text
Precision@K

Recall@K

MRR

NDCG

Hit Rate
```

MLflow **stores** these numbers.

It **doesn't calculate them**.

Example

```text
Evaluation Code

↓

Precision = 0.86

↓

mlflow.log_metric("precision",0.86)
```

So

Evaluation

↓

MLflow

Keep these as **four separate components** in your architecture:

1. **Evaluation** – Measures search quality using the WANDS benchmark.
2. **MLflow** – Records experiments, configurations, metrics, and artifacts.
